# CS 334 Final Project - Procedural Terrain Generator

This project is an infinite procedural terrain generator programmed in C++.

## Libraries used
- OpenGL + GLM + GLFW + Glad
- FastNoiseLite

## How to run
NOTE: This project is made for Mac only use
1. Run `cd build`
2. Run `make`
3. Run `./ProceduralTerrainGenerator`
